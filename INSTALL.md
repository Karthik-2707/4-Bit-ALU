# Installation Instructions for Verilog Simulation and Synthesis Tools

This document provides instructions for installing Verilog simulation and synthesis tools required for working with the 4-Bit ALU project.

## Tool Requirements

1. **Icarus Verilog**  
   - A popular open-source synthesize and simulation tool for Verilog.
   - Installation Command (Debian/Ubuntu):  
     ```bash
     sudo apt-get install iverilog
     ```

2. **GTKWave**  
   - A wave viewer for displaying simulation output. 
   - Installation Command (Debian/Ubuntu):  
     ```bash
     sudo apt-get install gtkwave
     ```

3. **Synopsys Design Compiler**  
   - Commercial synthesis tool (may require a license).

## Installation Instructions

### Step 1: Install Icarus Verilog

Use the following command to install Icarus Verilog on your system:

```bash
sudo apt-get install iverilog
```

### Step 2: Install GTKWave

To install GTKWave, execute:

```bash
sudo apt-get install gtkwave
```

### Step 3: Install Synopsys Design Compiler

For the Synopsys Design Compiler, please follow the specific installation instructions provided in the licensing package or your institution's guidelines.

## Conclusion
After completing these installations, you will be ready to simulate and synthesize the Verilog files in the 4-Bit ALU project.
